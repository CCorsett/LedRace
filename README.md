# Funny Led Race

Minimalist car race made with WS2812b LED. Similar to [led-race](https://github.com/gbarbarov/led-race) by gbarbarov.

## Installation and Usage

* Arduino uno REV 3
* WS2812b Led strip
* 5V 2.5A power alim
* Four arcade buttons
* Resistor, capacitor and 8 Ohm speaker
* Arduino IDE
* Adafruit neopixel library
